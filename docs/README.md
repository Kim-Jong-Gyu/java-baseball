### 기능 구현 목록 
- 입력 
    - 사용자에게 숫자를 입력 받는 기능
    - 사용자가 세 자리 수 이상을 입력할 경우 `IllegalArgumentException`을 발생시킨 후 발생시킨 후 종료 기능
- 기능
    - 1 ~ 9 서로 다른 임의수 3개의 숫자를 생성하여 3자리 숫자(숫자 배열)를 생성 기능
    - 스트라이크가 있는지 확인 기능
    - 볼이 있는지 확인 기능
    - 모두 맞히면 게임이 종료거나 다시 시작 기능
    - 게임을 시작한다.
- 출력
    - 입력한 수에 대한 결과를 출력 기능
